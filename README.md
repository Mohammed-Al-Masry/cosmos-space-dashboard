# COSMOS Space Dashboard

A visually appealing and interactive web application designed for space enthusiasts. It provides a clean and modern interface to explore information about the planets in our solar system. This project is built with fundamental web technologies and serves as a great example for beginner front-end developers.

## Project Overview

COSMOS Space Dashboard: A captivating frontend project built with HTML, CSS, and JavaScript. Explore the planets of our solar system through a sleek, user-friendly interface. Ideal for space enthusiasts and beginner frontend developers looking for a cool project.

## Features

- Interactive dashboard with details about the planets.
- Responsive design that works on different screen sizes.
- Visually engaging UI with beautiful planet imagery.
- Built with plain HTML, CSS, and JavaScript, making it lightweight and easy to understand.

## Technologies Used

- **HTML5:** For the structure of the web pages.
- **CSS3:** For styling and layout, including modern features for a better look and feel.
- **Font Awesome:** For icons used throughout the interface.
- **JavaScript (ES6+):** For interactivity and dynamic content.

## How to Use

To run this project locally, follow these simple steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/your-username/COSMOS-Space-Dashboard.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd COSMOS-Space-Dashboard
    ```
3.  **Open the `index.html` file in your browser:**
    You can do this by double-clicking the file or by right-clicking and selecting "Open with" your preferred web browser.

## Data Source

The data used in this project fetched from NASA's Open Data Portal, NASA APIs and "Launch Library 2" API from thespacedevs.com
